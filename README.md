# web_ph_2026

Un petit site pour augmenter la visibilité du PH et engager le travail d'archivage.
- `index.html` : landing page, mettre en forme d'un PH et mettre un gros bouton en mode "lire la dernière édition
- `archives.html` : liste de tous les précédents PH en format pdf sipossibl

Pour que les liens fonctionnent (fonction `fetch()`), il faut lancer un serveur http en faisant la commande suivante dans la racine du dossier :
```python
python -m http.server 8000
```
